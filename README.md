# PhoebeProject - PhoebeWebsite

Đây sẽ là website chính thức của CLB. Nghiêm cấm hành vi share, fork hay clone mà không xin phép bản quyền!!

## Hướng dẫn cài đặt

Sử dụng Git clone (đã cấp quyền contributor hay collaborator) trong VSCode.

```bash
git clone <link của repo>
```

## Usage

```bash
git init (khởi tạo)
git status (kiểm tra xem đang ở nhánh nào)
git add . (Thêm file vào nhánh chỉ định trong repo)
git commit (Thực hiện commit cho repo)

Tạo PR ở trên GitHub hoặc Terminal.
```

## Cách làm việc

Pull request miễn phí nên code đc tính năng gì hay hoặc fix đc bug nào thì tạo 1 cái. Ib riêng cho chủ thớt để tiến hành merge code rồi test (contributor hay collaborator phải test ở máy trước khi tạo pull request, thằng chủ chỉ test lại, kh fix lỗi hộ hoặc nhận code lỗi).
