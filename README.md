## Hướng dẫn cài đặt

Sử dụng Git clone (đã cấp quyền contributor hay collaborator) trong VSCode.

```bash
git clone <link của repo>
```

## Cách sử dụng

```bash
git init (khởi tạo)
git status (kiểm tra xem đang ở nhánh nào)
git add . (Thêm file vào nhánh chỉ định trong repo)
git commit (Thực hiện commit cho repo)

Tạo PR ở trên GitHub hoặc Terminal.
```
